# Mero Swasthya (FYP)

Mero Swasthya is a full-stack digital healthcare management system developed as a Final Year Project. It replaces manual hospital workflows with a centralized platform that integrates patient management, medical report processing, and billing.

---

## Features

- User authentication (login/signup with protected routes)
- Receptionist portal for patient registration and validation
- Doctor portal for managing patient records and prescriptions
- Pharmacy portal for medicine handling and billing
- Admin dashboard for system overview and control
- OCR-based extraction of medical report text
- NLP-based structured data processing
- Invoice and billing system

---

## Tech Stack

- Frontend: React.js
- Backend: Python (FastAPI)
- Database: PostgreSQL
- OCR & NLP: Custom-built processing modules

---

## Project Structure

- `app/` — Backend logic and API
- `client/` — Frontend React application
- `sample_reports/` — OCR test images
- `run.py` — Backend entry point

---

## Setup

```bash
pip install -r requirements.txt
python run.py
```

```bash
cd client
npm install
npm start
```
